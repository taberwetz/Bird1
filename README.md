# Flappy Penguin

A Flappy Bird clone starring a penguin, rendered in a pure black / white / orange palette.

## Play

Open `index.html` in any modern browser. No build step or dependencies.

## Controls

- **Space / Up Arrow** — flap
- **Click / Tap** — flap
- After a game over, tap or press space to restart.

## Files

- `index.html` — page shell and styling
- `game.js` — game loop, rendering, and input handling
